2015 MPark (Web App)
Description
Created during MHacks 6, a 36-hour hackathon at the University of Michigan, MPark is a web platform for renting parking spots. Drivers who need parking can search a location to find nearby parking spots for rent by owners. The screenshot shows a search for parking spots near "100 University Ave, Ann Arbor."
(Created with Shivani Shah).
Features
  Back-end integration with Parse's cloud database API
  Front-end integration Google Maps' Javascript API
  User interface built in HTML, CSS, and JQuery

2012 SushiDrop (Java Game)
Description
Created as a final project for AP Computer Science, SushiDrop is a game written in Java featuring a hamster that catches falling sushi while avoiding bottles of poison. The player uses the arrow keys to control a hamster on the left side of the window, while a computer player hamster competes on the right side.
(Created with Rachel Xiang).
Features
  Randomly generated falling objects (sushi and poison)
  Automated computer player that reacts to falling objects and includes three difficulty levels
  Increasing difficulty from start to finish of a round
  Collision detection
  Java GUI
